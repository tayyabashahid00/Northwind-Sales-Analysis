# 📊 Northwind SQL Analysis Using PostgreSQL

## Project Overview

This project demonstrates SQL-based sales analysis using the Northwind sample database in PostgreSQL. The objective is to analyze business data, answer real-world business questions, and gain insights into sales performance, customer behavior, product performance, and employee productivity.

The project showcases practical SQL skills commonly required for Data Analyst and Business Analyst roles.

---

## Objectives

The main objectives of this project are:

- Calculate the company's total sales revenue.
- Determine the total number of customers.
- Identify countries with the highest number of orders.
- Find the top 10 customers based on total sales.
- Identify the best-selling products.
- Analyze sales by product category.
- Evaluate employee performance based on processed orders.
- Analyze monthly order trends.
- Identify customers spending above the average.
- Rank customers using SQL window functions.
